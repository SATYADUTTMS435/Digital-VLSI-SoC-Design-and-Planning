# 🚀 Digital VLSI SoC Design and Planning

> **A comprehensive documentation of my six-week journey through the Digital VLSI SoC Design and Planning program by VLSI System Design (VSD), covering the complete RTL-to-GDSII ASIC Design Flow using industry-standard open-source EDA tools.**

---

# 📖 About the Course

Welcome to my documentation repository for the **Digital VLSI SoC Design and Planning** program organized by **VLSI System Design (VSD)**. This hands-on program is designed to provide practical exposure to the complete **RTL-to-GDSII ASIC Design Flow**, enabling participants to transform a Register Transfer Level (RTL) design into a manufacturable GDSII layout using industry-standard open-source EDA tools and the **SKY130 Process Design Kit (PDK)**.

The course emphasizes learning through **implementation, debugging, documentation, and engineering best practices**, closely resembling real semiconductor design workflows. Throughout the program, participants gain practical experience with synthesis, floorplanning, placement, clock tree synthesis (CTS), routing, static timing analysis (STA), power distribution network (PDN), and physical verification using the OpenLANE flow.

Unlike a conventional workshop submission, this repository is intended to serve as a **technical reference** containing detailed theory, implementation steps, screenshots, command explanations, report analysis, troubleshooting notes, observations, and interview-oriented insights.

---

# 🏛️ About VLSI System Design (VSD)

**VLSI System Design (VSD)** is an organization dedicated to bridging the gap between academic learning and semiconductor industry requirements through project-based education. Their programs focus on practical implementation using open-source EDA tools, allowing students and professionals to experience the complete ASIC design flow without requiring expensive commercial software.

The Digital VLSI SoC Design and Planning program introduces learners to the modern RTL-to-GDSII implementation flow using tools such as **OpenLANE**, **OpenROAD**, **OpenSTA**, **Yosys**, **Magic**, **Netgen**, and the **SKY130 PDK**, providing an excellent foundation for careers in Digital Physical Design.

# 🎓 Course Objectives

Throughout this program, I aim to:

- Understand the complete RTL-to-GDSII ASIC Design Flow.
- Learn the purpose and significance of every stage involved in Digital Physical Design.
- Gain practical experience using GitHub Codespaces and cloud-based development.
- Perform synthesis, floorplanning, placement, Clock Tree Synthesis (CTS), routing, Static Timing Analysis (STA), Power Distribution Network (PDN), and physical verification.
- Analyze reports, optimize timing, understand Engineering Change Orders (ECOs), and study signoff methodologies.
- Develop professional documentation practices suitable for engineering projects and collaborative development.
